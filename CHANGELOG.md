# Changelog

## 0.2.0
- Added an EMS mode Select for every individual battery.
- Added an individual battery power-control Number from -2000 W to +2000 W.
- Added support for `general`, `mqtt_ctrl`, and `tou_plan`.
- Added automatic 50-second power-command keepalive in MQTT control mode.
- Control entities are not created for the combined group device.
- Updated English and German translations and documentation.

## 0.1.5
- Changed combined battery-power sign convention: discharge negative, charge positive.
